# Data

The datasets are not included in this repository due to file size limitations.

## Download Instructions

### Walmart Store Sales Forecasting
1. Go to: https://www.kaggle.com/datasets/gustavoserafim/walmart-recruiting-store-sales-forecasting-gsr
2. Download and extract the CSV files
3. Place `stores.csv`, `features.csv`, and `train.csv` in the `walmart/` folder

### Online Retail II UCI
1. Go to: https://www.kaggle.com/datasets/mashlyn/online-retail-ii-uci
2. Download the CSV file
3. Place `online_retail_II.csv` in the `online_retail/` folder
